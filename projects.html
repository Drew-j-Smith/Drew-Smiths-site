Nintendo Switch Contoller Emulator

August 2020 C++ C Java Opencv Arduino

This was the most complicated project I have taken up until this now. The goal of this project was to be able to control a Nintendo Switch from a computer with the help of an Arduino Uno. My starting point was a demo of an arcade-style controller (Source). From there I did a lot of research into the architecture of the microcontroller and with the datasheet I enabled serial interface between the arduino and the computer. My goal from there was to be able to run image processing on the Switch and run macros accordingly.

My first implementation was in java. However for reasons I could not determine, it was very error prone. It was very unintuitive in how it was set up and new code had to be added if any changes were made to the image processing or the macros. It was for those reasons that I decided to scrap my original and start from sratch in c++. With all the information I had gathered from my first venture, I addressed the problems with my first implementation and added on key missing features such as macro recording and a simple way to add new image processing and macro functions. The final implementation can be seen in the video where the computer automatically plays the game Octopath Traveller.
